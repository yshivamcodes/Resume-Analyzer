## 🔥 Core Features

### 📊 Intelligent Resume Scoring

* Semantic similarity (TF-IDF + cosine similarity)
* Keyword alignment
* Skill overlap analysis

### 🧠 Explainable AI Insights

* Highlights missing skills
* Identifies weak resume sections
* Provides reasoning behind scores

### ✍️ AI-Powered Improvements

* Suggests actionable enhancements
* Rewrites resume bullet points professionally

### ⚡ Real-Time Analysis

* Upload resume + job description
* Get instant structured feedback

## 🚀 How It Works

1. Upload a resume (PDF)
2. Paste job description
3. System analyzes:

   * Skills
   * Keywords
   * Semantic similarity
4. Returns:

   * Match score
   * Missing skills
   * Improvement suggestions
